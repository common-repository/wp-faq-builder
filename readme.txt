== Description ==
Do you need any decent FAQ section which will be used to describe the details of your services, terms & conditions? You have a company and you want to make your users know about the history and background of your company in a Q&A format, then This plugin is for you. This WordPress FAQ plugin will be the most suitable tool for getting the highly professional result for your website.

WP FAQ Builder is a WordPress FAQ plugin that lets you create, organize and Advertise your FAQs (frequently asked questions) in no time through your WordPress admin panel. This plugin basically provides multiple layouts with a awesome styling options.

== Drag & Drop Facility ==

The FAQ Builder plugin allows you to apply drag and drop facility which makes the things more interactive and user friendly. You can shaffle the content elements according to your need. 
Easy Configuration

WP FAQ Builder ensures you an easy configuration that provides you all the solutions in a simple manner. This FAQ WordPress plugin allow you to create unlimited amount of questions and answers section regarding your company’s services and others aspects.

== Responsive Elegant Plugin ==
 WP FAQ Builder is 100% responsive means mobile friendly and compatible with all the modern browsers. This is the easy to access plugin which aim is to give you a comfortable way to interact with your clients or users.
Wonderful Styling Materials
There are various styling ingredients in the configuration part of the FAQ Plugin. If you want to customize the background of the template layouts. You will have 4 background structures along with different options.
You can change the question text color, answer text color and category text color as well.

== Multiple Layouts ==
There are 3 different layouts in this FAQ WordPress plugin. They play different unique functionalities.Here are the three layouts of the plugin:

**Accordion**
Accordion layout presents all the contents in a usual expandable accordion style which means only one panel can be expanded at any given time through each panel is equally expandable and collapsible.  

**List**
List layout present all the contents in a traditional list format one after one.

**Multicolumn**
Multicolumn usually presents all the contents through several column system.

== Installation ==

This section describes how to install the plugin and get it working.
e.g.

1. Upload the plugin files to the /wp-content/plugins/ directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the ‘Plugins’ screen in WordPress
1. Use the WP FAQ Builder -> -> Add FAQ List to create a table and get started.
